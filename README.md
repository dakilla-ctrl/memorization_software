#  Memorization Software - Memoware (Python GUI Version)
A text memorization tool built in Python using Tkinter. Users type out material segments to reinforce memory, with increasing difficulty levels and scoring

<img src="https://github.com/dakilla-ctrl/memorization_software/raw/python-version/assets/demo.gif" width="350" />


---

## Features

- **Custom Input**: Add your own material or text to memorize
- **Word Omission Practice**: Random blanks are created for you to fill in
- **Difficulty Scaling**: Higher levels increase blanks for deeper memory work
- **Instant Feedback**: Correct answers are highlighted and mistakes shown
- **Score Tracking**: Calculates your score after each attempt
- **Designed with **Students, Language learners and Catholics** in mind
